```
assets/
├── baua-logo.png          # Logo principal (pássaro com fundo verde-água)

```

## 🦜 Logo Principal (baua-logo.png)

**IMPORTANTE:** Salve a imagem do pássaro (fornecida) como `baua-logo.png` nesta pasta.

### Características da Logo:
- **Formato:** PNG com transparência
- **Dimensões recomendadas:** 512x512px ou maior
- **Fundo:** Verde-água (#1ABC9C) ou transparente
- **Descrição:** Silhueta de pássaro com capelo de formatura

### Como adicionar:
1. Salve a imagem do pássaro fornecida
2. Renomeie para `baua-logo.png`
3. Coloque nesta pasta (`frontend/src/assets/`)
